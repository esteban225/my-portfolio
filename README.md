## 🧪 Cómo Ejecutarlo Localmente

1. 📥 Clona el repositorio:
```bash
git clone https://github.com/esteban225/mi-portafolio.git
cd mi-portafolio
```

2. 📦 Instala las dependencias:
```bash
npm install
# o
yarn install
```

3. ▶️ Inicia el proyecto:
```bash
npm start
# o
yarn start
```

Abre `http://localhost:3000` en tu navegador 🚀
